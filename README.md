# Business Intelligence Student Project

This repository contains a business intelligence project with the following structure:

## Directory Structure

- **/data** - Contains datasets and data files used in the project
- **/notebooks** - Jupyter notebooks for analysis and exploration
- **/scripts** - Python scripts for data processing and analysis
- **/visuals** - Generated visualizations and charts
- **/report** - Project reports and documentation

## Getting Started

1. Clone this repository
2. Install required dependencies (if requirements.txt exists)
3. Start with the notebooks in the `/notebooks` directory

## Requirements

See requirements.txt for Python dependencies (optional)

## Usage

Each directory serves a specific purpose:
- Use `/data` to store your datasets
- Use `/notebooks` for interactive analysis
- Use `/scripts` for reusable code modules
- Use `/visuals` to save your visualizations
- Use `/report` for project documentation and reports
